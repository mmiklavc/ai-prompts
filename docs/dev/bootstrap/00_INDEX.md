# Polyglot Cursor Bootstrap Index

This collection defines how Cursor (or any AI-assisted coding tool) should reason about, edit, test, and validate code across a polyglot monorepo.

**Included stacks**
- Go
- Python
- Java (Maven)
- Bash/Shell
- Make
- Helm
- JSON
- gRPC / Protobuf

Each adapter enforces canonical best practices for that stack, aligned with official style guides and static-analysis tooling.
